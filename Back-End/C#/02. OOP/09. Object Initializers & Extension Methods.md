Object Initializers allow you to create an object and set its properties **in one expression** without needing a constructor.

```CS
// Without Object Initializer
Person p = new Person();
p.Name = "Hamza";
p.Age = 22;

// With Object Intializer
Person p = new Person {
    Name = "Hamza",
    Age = 22
};

// With Constructor + Object Initializer
Person p = new Person("Hamza") {
    Age = 22,
    City = "Cairo"
};
```

Extension Methods let you **add new methods to an existing class without editing the class itself**.

**How to create an Extension Method**:
1. Create a **static class**.
2. Create a **static method**.
3. First parameter must start with `this` and the type you want to extend.

```CS
// Example: Add a new method to String class
public static class StringExtensions {
    public static bool IsLong(this string str){
        return str.Length > 10;
    }
}

string s = "Hello there!";
bool result = s.IsLong();  // True
```
