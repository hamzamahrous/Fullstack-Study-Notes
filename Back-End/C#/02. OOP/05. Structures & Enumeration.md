Struct is **value type** unlike the classes.

```CS
struct Point {
    public int X;
    public int Y;
}

var p1 = new Point { X = 5, Y = 10 };
var p2 = p1;   // copies the values

p2.X = 20;

Console.WriteLine(p1.X); // still 5
```

| Feature             | Class           | Struct                              |
| ------------------- | --------------- | ----------------------------------- |
| Type                | Reference       | Value                               |
| Memory location     | Heap            | Stack (usually)                     |
| Assignment          | Copy reference  | Copy value                          |
| Inheritance         | Yes             | No (only interfaces)                |
| Default constructor | Allowed         | Not allowed to define parameterless |
| Best use-case       | Complex objects | Small data objects                  |

An enum is basically a **way to give names to numbers**.

```CS
enum OrderStatus{
    Pending,    // 0
    Shipped,    // 1
    Delivered,  // 2
    Cancelled   // 3
}

OrderStatus status = OrderStatus.Shipped;

// Print Numeric Value
Console.WriteLine((int)status); // prints 1

// Changing enum values manually
enum ErrorCode{
    NotFound = 404,
    ServerError = 500,
    Unauthorized = 401
}
```
